NOTES FOR GETTING STARTED WITH HTML

World Wide Web
A collection of HTML documents

HTML (Hypertext Markup Language)
Contains text content, markup, references, links

Hyperlinks
Links between HTML pages

HTTP
Main protocol of the web

Servers
Computers that host files that make up the web

Internet
World's largest computer network

Browser
Program that runs on your ocmputer to display files found on the web

HTML Tags
Elements - "<" Name ">" Content "<" "/" Name ">"
Examples: b - bold, em - italics

HTML Attributes
anchor - a (used for links, attribute: href)
images - img (used for images, attribute: src)
Multiple Lines
break - br
paragraph - p

Inline:
Examples: a, img, br, strong, span

Block: forms an "invisible box"
Examples: div, p

Additional Notes

All HTML tags and other actions can found by searching online
Programmers do not need to memorize everything
